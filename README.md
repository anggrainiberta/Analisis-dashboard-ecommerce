# E-Commerce Analytics Dashboard

Dashboard Analisis E-Commerce Menggunakan Python & Streamlit.

## Deskripsi

E-Commerce Analytics Dashboard merupakan dashboard interaktif yang digunakan untuk menganalisis perilaku pelanggan dan performa penjualan e-commerce menggunakan pendekatan Exploratory Data Analysis (EDA) dan K-Means Clustering.

## Fitur

- Analisis Distribusi Pelanggan (Usia, Gender, Negara)
- Tren Revenue Harian & Bulanan
- Analisis Traffic Source
- Segmentasi Pelanggan (K-Means Clustering)
- Segmentasi Kategori Produk
- Filter Interaktif (Tahun, Gender, Negara)
